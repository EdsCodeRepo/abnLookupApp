Problem statement: 

	A small tool is required to extract ABN data from their webpage 
	the publicly available data base, format the data and save to a
 	database in a structured format.


Design requirements:

	Accept ABN as input into CLI or a config file (perhaps,
	secondary pathway might be useful or bloat, will consider)

	Utilize browser automation (likely playwright)

	Must navigate to ABN lookup, preform search using users supplied ABN and 
	extract Entity name, ABN status (Active/Cancelled), entity type and save result
	formatted as a JSON file.

	Handle errors gracefully, specifically invalid ABN cases and page selector not 	found (check playwright docs otherwise can build it into the script)

	
	Log page metadata, access timestamp (other important info will be come apparent) 	with the option to return associated info when querying saved ABN details
	
	Containerise application with docker, will simplify setup instructions (need to 	look into industry practice using docker, see docs)


	Include documentation with 
		explicit setup instructions (environment reproduction)
		How to operate the script (CLI or config instructions)
		Any assumptions made 

		

Design characteristics 
	
	Python based automation script using browser automating tooling (likely playwright) 	that will do most of the heavy lifting 

	Main script will be the databasing, accessing and logging (plus error handling 	playwright doesn't catch/edge cases)

	
	Intitial user plane can let them choose headless/open a browser in the background
